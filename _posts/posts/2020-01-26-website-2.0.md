The last website I made turned out to be a bit too much for me. Too many fancy features and dynamic moving graphics.

![old-website image]({{site.baseurl}}/images/website-2.0/old-website.png)

So this is my round two for personal website. Using [Jekyll](https://jekyllrb.com/) really made the process simple for me, especially the ability to host the website on my local machine for rapid testing. Another super nice feature of Jekyll is the the ability to write all posts in markdown. That way I don't have to mess with all the messy html tags which saves me lots of time. Finally, there are plenty of free [Jekyll themes](http://jekyllthemes.org/) available online, so any web noob like me could get started on on personal website in minutes :)
